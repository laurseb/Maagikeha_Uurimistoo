Sellesse Github repository'sse on kogutud Risto Lauri uurimistöö "Kirde-Eesti magnetvälja ja raskusjõu anomaaliad kui võimalike maagikehade peegeldused" tarvis loodud mudeldamise ja analüüsi programmid. Lisatud programmide loomiseks on kasutatud tehisaru ning väliseid näiteid.

"maagikeha3d.py" - Kolmedimensioonilise maagikeha mudel, mis aruvtab kasutaja poolt seatud parameetritega maagikeha väljade avaldumise maapinnale. 

"anomaaliateandmeextract.py" - Programm, mis arvutab mudeli tulemustest lisaandmeid, et oleks võimalik andmestikuga võrrelda.

"anomaalialeidja.py" - Programm, mis filtreerib antud gravi- ja magnetomeetrilistest andmesikest anomaalsed piirkonnad ning võrdleb mudeldatud maagikeha parameetritega. Arvutab lisaks ka korrelatsiooni andmestik anomaalia ja mudeli vahel.
